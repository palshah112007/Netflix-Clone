# Netflix Clone 🎬

A professional Netflix-inspired web application built with HTML5, CSS3, and JavaScript.

## Project Structure 📁

```
netflix-clone/
├── index.html              # Main entry point
├── netflix-logo.webp       # Netflix logo
├── assets/                 # Static files
```

## Features ✨

### Header/Navigation
- Fixed navigation bar with Netflix logo
- Browse menu (Home, TV Shows, Movies, etc.)
- Search functionality
- User profile dropdown menu
- Navbar changes appearance on scroll

### Hero Section
- Full-screen featured content display
- Beautiful background image with overlay
- Call-to-action buttons (Play, More Info)
- Responsive design

### Content Sections
- **Trending Now** - Currently popular shows
- **Action** - Action movies and series
- **Drama** - Drama content
- **Comedy** - Comedy series and specials
- **Recently Added** - New releases

### Movie Cards
- Hover effects with zoom animation
- Movie overlay with title and ratings
- HD/4K quality indicators
- Star ratings display
- Smooth scrolling carousel

### Footer
- Multiple link sections (Company, Help, Legal, Social)
- Professional layout
- Responsive columns

### Interactive Features
- Scroll-triggered navbar styling
- Carousel drag functionality
- Profile dropdown menu
- Search bar with focus animation
- Hover effects on all interactive elements

## How to Use 🚀

1. **Open in Browser**
   - Simply open `index.html` in your web browser
   - No server setup required for basic viewing

2. **Navigation**
   - Click on menu items to navigate (demo only)
   - Use search box to search (demo functionality)
   - Hover over movie cards for details
   - Click profile icon for dropdown menu

3. **Customize**
   - Replace movie URLs with real image links
   - Update movie titles and descriptions
   - Modify colors in CSS variables
   - Add real Netflix links

## Responsive Design 📱

The website is fully responsive and optimized for:
- Desktop (1920px and above)
- Laptop (1200px and above)
- Tablet (768px and above)
- Mobile (480px and above)

Breakpoints:
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: 480px - 767px
- Small Mobile: Below 480px

## Customization Tips 💡

### Change Colors
Edit the `:root` variables in `css/style.css`

### Add New Sections
Copy any `.section-container` div and modify the content

### Modify Animations
Edit transition and transform properties in CSS

### Add Real Data
Replace placeholder images with real URLs
Update movie titles and descriptions
